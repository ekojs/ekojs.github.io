Artikel ini pada akhirnya saya post sekarang, karena banyak permasalahan mengenai keamanan data pada harddisk di komputer PC ato Laptop berbasis Windows. Permasalahan muncul ketika ada seseorang entah usil atau hal yang lain yang mencuri data menggunakan flashdisk secara sengaja saat kita tidak sedang berada di depan PC/Laptop, bisa jadi memang teman atau seseorang ingin meminjam PC/Laptop untuk keperluan buka internet, tapi bukannya untuk baca artikel internet malahan mencuri data.


Oleh karena itu, apapun kondisi dan keperluannya bila berkenaan dengan COPY DATA, anda sendiri yang harus turun tangan, terkecuali anda percaya pada siapa anda pinjamkan PC/Laptop anda. Bila tidak, APLIKASI ini cocok buat anda untuk mengamankan data anda.



Cara kerja aplikasi ini cukup mudah, anda cukup mengaktifkan fitur Write Protection ON, agar segala macam device sperti flashdisk dkk, tidak dapat mengakses data anda. Untuk mematikan cukup di OFF saja.



Prinsip kerja aplikasi ini simpel, anda harus melepas segala flashdisk dkk sebelum mengON/OFFkan fitur itu, Jangan mengON/OFFkan saat device lain sperti flashdisk dkk masih terpasang pada PC/Laptop anda !!!

Aplikasi ini saya buat dengan menggunakan Console Script pada Windows OS, dan saya compile menjadi executable file..
