This application intended to prevent malicious code like virus to change or corrupt an office application or another file in windows system. This application will rename an extensions of list file from path you give. Here's the brief explanation :


  1. Give your folder path where you'll rename your extensions of file... Ex :"D:\test"
  2. Input your original extension of that file... Ex : "txt"
  3. Input your new extension of that file.... Ex : "eko_txt"
  4. After that step by step process you'll get a log from file you have been renamed...
  5. Finished...
