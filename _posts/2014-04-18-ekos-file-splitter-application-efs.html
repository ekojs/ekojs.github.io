Aplikasi ini hampir mirip dengan aplikasi HJSplit yang mungkin telah anda kenal sebelumnya. Tapi aplikasi ini memiliki fitur untuk menggabungkan file sumber dengan file lain semisal file gambar. Fitur ini adalah Steganography yang ada pada aplikasi ini. Tapi, aplikasi ini juga masih memiliki kekurangan, dan bisa anda perhatikan sendiri nantinya bila anda mencoba aplikasi ini.

Ini adalah tampilan saat men-split file video, untuk ukuran file bila kurang dari 1 MB, maka gunakan pemisah tanda koma misalkan anda menginginkan mempartisi file sebesar 500KB, maka anda harus menuliskan 0,5 MB.

Ini adalah output dari hasil split file diatas, memiliki format nama_fileN.ekofs. Perbedaan output dari EFS dan HJSplit ini adalah format output yang diberikan. Untuk HJSplit format yang diberikan berupa :
  1. nama_file.ekstensi.001  --> ini adalah format HJSplit
  2. nama_file1.ekofs  --> ini adalah format EFS
Jadi, bila ingin menggabungkan file dari hasil output HJSplit anda harus mengubah formatnya terlebih dahulu. Untuk ekstensi pada EFS memang sengaja tidak disertakan karena pada dasarnya aplikasi ini digunakan untuk keamanan dari data yang dipartisi agar orang lain tidak tahu apa ekstensi file itu.

Untuk penggunaan tombol "Merge With" itu digunakan untuk menggabungkan file partisi dengan file lain misal gambar. Jadi bila partisi file itu kita gabung dengan file JPG, maka ekstensi file partisi itu akan memiliki ekstensi JPG juga. seperti yang ditunjukkan pada gambar dibawah ini.

Setiap penggabungan file partisi dengan file lainnya, di dalam file "KEYFILE" akan terdapat key file yang harus dimasukkan kedalam aplikasi untuk menggabungkan kembali file partisi tersebut, bila tidak file akan kacau dan MD5 outputnya tidak akan sama dengan MD5 yang sebenarnya. Perhatikan gambar diatas.

Proses partisi dan penggabungan data membutuhkan waktu yang kadang tak sama ditiap komputer yang telah dicoba, ketika proses 100% selesai aplikasi akan diam sejenak untuk mem-flush cache pada komputer.
Jadi anda hanya tinggal menunggu pesan konfirmasi bahwa file telah selesai di partisi / di gabungkan.
